    Beginner guide

    Statit is a tool to access metrics easily. This guide will explain you the basics.

    This guide is organized in three parts:

    • Getting started - Understanding how Statit works
    • Using the website - Creating an account, favorites & alerts and dashboards
    • Using the spreadsheet plugins - Accessing metrics directly in either Excel or Google Sheets
